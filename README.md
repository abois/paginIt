paginIt
=======

A light jQuery plugin for pagination


Options
-------

matchingElement
  The elements to match for pagination
  default: 'article'
width
 The elements width
  default: '800'
prevContent
 The prev button html content
 default: '&lt;'
nextContent
  The next button html content
  default: '&gt;'
cssPrefix
  The prefix for css classes
  default: 'paginit-'
effectDuration: 0
  The show effect duration
  default: 0
